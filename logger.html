CAN1 — OBD 3/11, 500 kbps — radio/SYNC communication and the 0x2B4 vehicle-data message.
CAN2 — OBD 6/14, 500 kbps — main engine/PCM diagnostics and UDS requests such as 7E0/7E8.
CAN3 — OBD 1/8, 125 kbps — alternate network for other or fallback vehicle modules; its exact role depends on vehicle configuration.

Before sending a test request, start the CAN bus that contains its target module and set a filter that includes the expected response ID.
Engine/PCM parameters — start CAN2 and use exact filter 7E8; use range 7E8–7EC when checking several diagnostic modules.
TPMS and battery presets — start CAN2 and filter 72E.
Distance DID 404C — start CAN1 and filter 728.
Select All IDs if the response address is unknown, then narrow the filter after identifying it.
